# kleiderschrank

Pronomen und Namen ausprobieren.

Das hier ist eine Sammlung statisches HTML, CSS, Javascript und ein paar sehr exzellente Texte. Zusammen ergeben sie den Pronomenkleiderschrank, der ermöglicht, Namen und Pronomen auszuprobieren und zu üben.

Wahrscheinlich brauchst du dieses Repo nicht, der Kleiderschrank ist live auf https://kleiderschrank.queer-lexikon.net

## Installation

schnapp dir alles hier drin und werf es in ein Document-Root. Yay.

## Danksagungen

Das Kleiderschrank-Projekt wurde inspiriert durch den englischen Pronoun Dressing Room: http://www.pronouns.failedslacker.com/.

Das Queer Lexikon hat diesen mit einer Projektförderung der deutschen Postcode-Lotterie umgesetzt. Das Design kommt von Kim und Kim: https://owlisdoodles.art/, die Texte von Valo: https://valochristiansen.com.

Wir nutzen Icons von [Font Awesome](https://fontawesome.com).
