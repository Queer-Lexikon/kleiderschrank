# kleiderschrank

Pronomen und Namen ausprobieren.

Das hier ist eine Sammlung HTML, CSS, Javascript und ein paar sehr exzellente Texte (abweichend lizensiert unter CC-BY-SA 4.0 Queer Lexikon). Zusammen ergeben sie den Pronomenkleiderschrank, der ermöglicht, Namen und Pronomen auszuprobieren und zu üben.

Wahrscheinlich brauchst du dieses Repo nicht, der Kleiderschrank ist live auf https://kleiderschrank.queer-lexikon.net.

## BTD: Bauen, testen und deployen

`npm install`
`npm run build`
`npm run lint`
`npm test`

Schnapp dir danach alles in `dist/` und wirf es in ein Document-Root. Yay.

## Entwickeln

Du kannst in diesem Verzeichnis einen Webserver starten (z.B. mit `python3 -m http.server`) und `./index.html` aufrufen, ohne irgendwas komplieren oder bauen zu müssen. Das geht prinzipiell auch ohne node, aber wir empfehlen wegen der Größe und der ungeahnten Komplexitäts dieses Projekts dringend, Unittests zu bauen und ESLint zu benutzen.

## Danksagungen

Das Kleiderschrank-Projekt wurde inspiriert durch den englischen Pronoun Dressing Room: http://www.pronouns.failedslacker.com/.

Das Queer Lexikon hat diesen mit einer Projektförderung der deutschen Postcode-Lotterie umgesetzt. Das Design kommt von Kim und Kim: https://owlisdoodles.art/, die Texte von Valo: https://valochristiansen.com.

Wir haben auch mit einer nicht näher benannten User Experience-Designerin zusammengearbeitet, um das Tool so einfach und robust nutzbar wie möglich hinzubekommen.
